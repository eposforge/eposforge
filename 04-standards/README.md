## Superseded standards

Superseded standards are removed from the active `04-standards/` tree to keep the directory readable (see the rule in `00-standards-meta/standards-meta.md`). Supersession is recorded in the replacement standard's `Status` section. The full prior content remains available in git history.

Additional standards are added here as they are adopted.
